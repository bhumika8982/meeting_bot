AI Meeting Assistant (MOM Bot)An automated meeting assistant designed to join Zoom meetings, record high-quality system audio, transcribe conversations with speaker identification, and generate structured Minutes of Meeting (MOM) using advanced AI.
1. Architecture & ApproachThe project follows a modular architecture consisting of three main engines:Automation Engine (Selenium): Handles the web-based Zoom interface, including entering meeting IDs, passcodes, and managing UI-level muting/joining.Audio Recording Engine (Soundcard & Numpy): Captures high-fidelity system audio through a loopback interface, ensuring that both the host and participants' voices are recorded clearly in Mono WAV format.AI Processing Engine (Deepgram & OpenAI): * Deepgram (Nova-2): Provides fast, accurate transcription with Diarization (detecting who said what).OpenAI (GPT-4o-mini): Processes the raw transcript to extract key insights, summaries, and action items.
 2. Setup & InstallationSince this is a Local MVP (Minimum Viable Product), follow these steps to run it on your machine:Clone the Repository:Bashgit clone https://github.com/bhumika8982/meeting_bot.git
cd meeting_bot
Install Requirements:Bashpip install django selenium deepgram-sdk==3.* openai soundcard soundfile numpy

Web Driver: Ensure chromedriver.exe (matching your Chrome version) is placed in the bot/ directory.Run Server:Bashpython manage.py runserver
3. API DocumentationThe backend exposes the following internal endpoints for the Django frontend:EndpointMethodDescription/api/bot/start/POSTInitiates the Selenium bot and starts audio recording./api/bot/stop/POSTSafely leaves the meeting and triggers AI processing./api/bot/results/GETFetches the generated transcript and MOM files.
